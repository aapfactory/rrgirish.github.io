PowerActions for the vSphere Web Client is one interesting fling was released by VMware recently and is a nice marriage of the UI with the CLI.
This plugin for the Web Client allows you to run the powerCLI scripts that you wrote using a context menu. It even has a powerCLI console displaying right inside your web client! How cool is that!
More details can be found at the VMware blogs:
http://blogs.vmware.com/PowerCLI/2014/09/poweractions.html
