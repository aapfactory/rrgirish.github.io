So it looks like the plugin doesn’t work on Chrome and IE11 browsers. Installing it multiple times continues to show that it is not installed. However, there is a workaround for Chrome which is as follows:
  1. Download the plugin.
  2. Close the browser.
  3. Run the plugin installer as administrator. (Right click->Run as administrator)
Now the client plugin works.
